Практическая работа по WEB

1. Ssh подключение

Подключение к учебному серверу помощью команды ssh.

2. Ping kubsu.ru

Проверка доступности сервера kubsu.ru с помощью команды ping.

3. Nslookup (A и MX записи)

Определение A-записей и MX-записей доменов kubsu.ru и kubsu-dev.ru.

4. Whois (дата регистрации доменов)

Получение информации о регистрации доменов kubsu.ru и kubsu-dev.ru.

5. Git clone

Клонирование репозитория с GitHub на учебный сервер с помощью команды git clone

6. SFTP скачивание файлов

Скачивание файлов с сервера по протоколу SFTP.
